PCA and the standard autoencoder are unsupervised dimensionality reduction methods, and their learned features are not discriminative. If you build a classifier upon the low-dimenional features extracted by PCA and autoencoder, you will find the classification accuracy very poor.

Linear discriminant analysis (LDA) is a traditionally supervised dimensionality reduction method for learning low-dimensional features which are highly discriminative. Likewise, can we extend autoencoder to supervised leanring?

import numpy

def to_one_hot(y, num_class=10):
    results = numpy.zeros((len(y), num_class))
    for i, label in enumerate(y):
        results[i, label] = 1.
    return results

# build the supervised autoencoder network
from keras.layers import Dense, Input, Dropout
from keras import models

input_img = Input(shape=(784,), name='input_img')

# encoder network
encode0 = Dropout(0.5)(input_img)
encode0 = Dense(384, activation='relu', name='encode0')(encode0)
encode1 = Dense(128, activation='relu', name='encode1')(encode0)
encode1_5 = Dense(64, activation='relu', name='encode2.5')(encode1)
encode2 = Dense(32, activation='relu', name='encode2')(encode1_5)
encode3 = Dense(8, activation='relu', name='encode3')(encode2)

# The width of the bottleneck layer must be exactly 2.
bottleneck = Dense(2, activation='relu', name='bottleneck')(encode3)


# decoder network
decode1 = Dense(8, activation='relu', name='decode1')(bottleneck)
decode2 = Dense(32, activation='relu', name='decode2')(decode1)
decode2_5 = Dense(64, activation='relu', name='decode2.5')(decode1)
decode3 = Dense(128, activation='relu', name='decode3')(decode2_5)
decode3_5 = Dropout(0.5)(decode3)
decode3_5 = Dense(384, activation='relu', name='decode3.5')(decode3_5)
decode3_5 = Dropout(0.75)(decode3_5)
decode4 = Dense(784, activation='relu', name='decode4')(decode3_5)
# build a classifier upon the bottleneck layer

classifier1 = Dense(8, activation='relu', name='classifier1')(bottleneck)
classifier2 = Dense(8, activation='relu', name='classifier2')(classifier1)

#<Add more dense layers and regularizations...>
classifier3 = Dense(10, activation='softmax', name='classifier3')(classifier2)
